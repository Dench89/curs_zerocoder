KodaCode — AI-ассистент для разработки, интегрирующийся в IDE и CLI. Работает в агентском режиме: читает файлы, пишет и редактирует код, запускает команды в терминале.

Контекст — совокупность данных, которые передаются модели для генерации ответа: файлы, код, история диалога, документация. Определяет качество результата.

IDE (Integrated Development Environment) — интегрированная среда разработки. Примеры: Visual Studio Code, PyCharm, PHPStorm.

CLI (Command Line Interface) — интерфейс командной строки. Способ взаимодействия с программой через текстовые команды в терминале.

Итеративная разработка — подход, при котором проект создаётся поэтапно: каждая итерация добавляет новый слой функциональности и даёт рабочий результат.

MVP (Minimum Viable Product) — минимально жизнеспособный продукт. Версия с базовым набором функций, достаточная для проверки идеи.

requirements.txt — файл в Python-проектах, фиксирующий список используемых библиотек и их версий. Обеспечивает воспроизводимость окружения.

.gitignore — файл конфигурации Git, в котором перечислены файлы и папки, которые система контроля версий должна игнорировать.

venv (virtual environment) — виртуальное окружение Python. Изолированное пространство с собственными зависимостями, не влияющими на глобальный Python.

TDD (Test-Driven Development) — разработка через тесты. Подход, при котором сначала пишутся тесты, а потом — код, который им соответствует.

Flask — лёгкий Python-фреймворк для создания веб-приложений.

argparse — встроенная Python-библиотека для разбора аргументов командной строки.

JSON (JavaScript Object Notation) — формат хранения и передачи данных в виде текста. Используется как база данных в нашем проекте.

Git — система контроля версий, позволяющая отслеживать изменения в коде и откатываться к предыдущим версиям.

CI/CD (Continuous Integration / Continuous Deployment) — практика автоматизации сборки, тестирования и деплоя кода.

DevOps — набор практик, объединяющих разработку и эксплуатацию программного обеспечения с целью ускорения поставки продукта.
