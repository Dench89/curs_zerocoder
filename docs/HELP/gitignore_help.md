Перед тем как делать коммит в репозиторий, убедитесь, что в проекте есть файл .gitignore. В нём прописывается, какие файлы Git будет игнорировать.

Что нужно добавить в .gitignore:

папку `venv` (виртуальное окружение)
`__pycache__`
`.pytest_cache`
локальные JSON-файлы с реальными данными
Зачем это нужно:

Уменьшает размер репозитория — папка `venv` со всеми модулями может весить от 500 МБ и более
Исключает утечки данных — персональные данные не попадут даже в первый коммит
Предотвращает конфликт окружений — если не исключить `venv`, у другого разработчика при клонировании возникнет конфликт окружений
