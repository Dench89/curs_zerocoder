Навигация
Команда	Назначение
pwd	Показать текущую папку.
ls	Показать содержимое папки.
ls -l	Подробный список файлов.
ls -la	Показать все файлы, включая скрытые.
cd ПАПКА	Перейти в папку.
cd ..	Перейти на уровень выше.
cd ~	Перейти в домашнюю папку.
clear	Очистить экран терминала.

Файлы и папки
Команда	Назначение
mkdir ПАПКА	Создать папку.
rmdir ПАПКА	Удалить пустую папку.
rm ФАЙЛ	Удалить файл.
rm -r ПАПКА	Удалить папку с содержимым.
rm -rf ПАПКА	Принудительно удалить папку.
cp ФАЙЛ КУДА	Копировать файл.
cp -r ПАПКА КУДА	Копировать папку.
mv ФАЙЛ КУДА	Переместить или переименовать файл.
touch ФАЙЛ	Создать пустой файл.

Просмотр файлов
Команда	Назначение
cat ФАЙЛ	Показать содержимое файла.
less ФАЙЛ	Постраничный просмотр файла.
head ФАЙЛ	Показать начало файла.
tail ФАЙЛ	Показать конец файла.
tail -f ФАЙЛ	Следить за изменением файла в реальном времени.

Поиск
Команда	Назначение
find ПАПКА -name "ИМЯ"	Найти файл.
locate ИМЯ	Быстрый поиск файла.
grep "ТЕКСТ" ФАЙЛ	Найти текст в файле.
grep -r "ТЕКСТ" ПАПКА	Найти текст во всех файлах папки.
which ПРОГРАММА	Показать путь к программе.

Права доступа
Команда	Назначение
chmod 755 ФАЙЛ	Изменить права доступа.
chmod +x ФАЙЛ	Сделать файл исполняемым.
chown USER:GROUP ФАЙЛ	Изменить владельца файла.
ls -l	Посмотреть права доступа.

Пользователи
Команда	Назначение
whoami	Показать текущего пользователя.
id	Показать UID и группы пользователя.
passwd	Сменить пароль.
sudo КОМАНДА	Выполнить команду от имени администратора.
su -	Переключиться на пользователя root.

Процессы
Команда	Назначение
ps aux	Показать все процессы.
top	Мониторинг процессов.
htop	Улучшенный мониторинг процессов.
kill PID	Завершить процесс.
kill -9 PID	Принудительно завершить процесс.

Система
Команда	Назначение
hostname	Показать имя компьютера.
hostnamectl	Показать информацию о системе.
uname -a	Показать информацию о ядре.
df -h	Использование дисков.
du -sh ПАПКА	Размер папки.
free -h	Использование оперативной памяти.
uptime	Время работы системы.

Сеть
Команда	Назначение
ip a	Показать сетевые интерфейсы.
ip r	Показать маршруты.
ping АДРЕС	Проверить доступность узла.
ss -tulpn	Показать открытые порты.
curl URL	Выполнить HTTP-запрос.
wget URL	Скачать файл.

SSH
Команда	Назначение
ssh user@host	Подключиться по SSH.
scp ФАЙЛ user@host:/путь	Копировать файл по SSH.
rsync -av ПАПКА user@host:/путь	Синхронизировать папку.

Службы (systemd)
Команда	Назначение
systemctl status СЛУЖБА	Проверить состояние службы.
systemctl start СЛУЖБА	Запустить службу.
systemctl stop СЛУЖБА	Остановить службу.
systemctl restart СЛУЖБА	Перезапустить службу.
systemctl enable СЛУЖБА	Включить автозапуск.
systemctl disable СЛУЖБА	Отключить автозапуск.

Логи
Команда	Назначение
journalctl	Просмотреть системный журнал.
journalctl -xe	Последние ошибки системы.
journalctl -u СЛУЖБА	Логи службы.
dmesg	Сообщения ядра Linux.

Пакеты (ALT Linux)
Команда	Назначение
apt update	Обновить список пакетов.
apt upgrade	Обновить установленные пакеты.
apt install ПАКЕТ	Установить пакет.
apt remove ПАКЕТ	Удалить пакет.
apt search ПАКЕТ	Найти пакет.

Архивы
Команда	Назначение
tar -czf archive.tar.gz ПАПКА	Создать архив.
tar -xzf archive.tar.gz	Распаковать архив.
zip -r archive.zip ПАПКА	Создать ZIP-архив.
unzip archive.zip	Распаковать ZIP-архив.
